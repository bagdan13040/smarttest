# SmartTest Design Guidelines

Этот документ описывает основные принципы дизайна, цветовую палитру, типографику и компоненты интерфейса приложения **SmartTest**. Используйте этот гайдлайн при добавлении новых экранов или изменении существующих, чтобы сохранить визуальную целостность проекта.

## 1. Цветовая палитра (Color Palette)

Приложение использует светлую, "теплую" тему с ярким акцентным синим цветом. Все цвета задаются в формате RGBA (0-1).

### Основные цвета
| Название | RGBA | Hex (прим.) | Использование |
| :--- | :--- | :--- | :--- |
| **Primary Blue** | `0.15, 0.55, 0.9, 1` | #268CE6 | Основные кнопки, активные иконки, заголовки, акценты. |
| **Background** | `0.95, 0.93, 0.90, 1` | #F2EDE6 | Основной фон экранов (теплый оттенок). |
| **Surface White** | `1, 1, 1, 1` | #FFFFFF | Фон карточек, полей ввода, панелей навигации. |
| **Text Primary** | `0.2, 0.2, 0.2, 1` | #333333 | Основной текст, заголовки карточек. |
| **Text Secondary** | `0.5, 0.5, 0.5, 1` | #808080 | Подписи, неактивные элементы, второстепенный текст. |

### Статусные цвета
| Название | RGBA | Использование |
| :--- | :--- | :--- |
| **Success Green** | `0.3, 0.8, 0.4, 1` | #4CCC66 | Успешные действия, правильные ответы. |
| **Error Red** | `0.8, 0.35, 0.35, 1` | #CC5959 | Ошибки, удаление, неправильные ответы. |
| **Warning/Diff** | `0.9, 0.6, 0.2, 1` | #E69933 | Средняя сложность, предупреждения. |

## 2. Типографика (Typography)

Используются системные шрифты (Roboto/San Francisco по умолчанию в Kivy). Размеры задаются в `sp` (scale-independent pixels).

*   **H1 (Заголовки экранов):** `22sp` - `24sp`, Bold. Цвет: Primary Blue.
*   **H2 (Заголовки карточек/секций):** `18sp`, Bold. Цвет: Text Primary.
*   **Body (Основной текст):** `16sp`. Цвет: Text Primary.
*   **Caption/Label (Подписи):** `14sp`. Цвет: Text Secondary.
*   **Button Text:** `16sp` - `20sp`, Bold. Цвет: White (на Primary фоне) или Primary (на светлом фоне).

---

## 3. Компоненты UI (UI Components)

### Кнопки (RoundedButton)
Основной элемент взаимодействия.
*   **Форма:** Скругленные углы, радиус `dp(14)` - `dp(20)`.
*   **Фон:** Primary Blue (для основных действий).
*   **Текст:** Белый, Bold, по центру.
*   **Реализация:** Используется кастомный класс `RoundedButton` с отрисовкой `RoundedRectangle` в `canvas.before`.

### Карточки (Cards)
Используются для отображения курсов, тем и блоков информации.
*   **Фон:** Surface White.
*   **Скругление:** `dp(16)`.
*   **Отступы:** Внутренние `dp(12)` - `dp(16)`.
*   **Тень/Граница:** Легкая тень или отсутствие границы (flat style) на контрастном фоне.

### Поля ввода (TextInput)
*   **Фон:** Белый или очень светло-серый (`0.98, 0.98, 0.98, 1`).
*   **Скругление:** Стандартное или `dp(10)`.
*   **Padding:** `dp(10)`.
*   **Cursor Color:** Primary Blue.

### Навигация (Bottom Navigation)
*   **Высота:** `dp(64)`.
*   **Фон:** Белый с верхней границей (`0.92, 0.92, 0.92, 1`).
*   **Иконки:** `34dp` x `34dp`.
*   **Состояние:**
    *   *Active:* Иконка Primary Blue, фон кнопки Primary Blue (с альфа 0.15).
    *   *Inactive:* Иконка Grey, фон прозрачный.

---

## 4. Отступы и Размеры (Layout & Spacing)

Все размеры должны использовать `dp` (density-independent pixels) для корректного отображения на разных экранах (особенно Android).

*   **Screen Padding:** `dp(16)` (стандартный отступ от краев экрана).
*   **Element Spacing:** `dp(10)` - `dp(12)` (расстояние между элементами в списке).
*   **Section Spacing:** `dp(20)` - `dp(32)` (расстояние между крупными блоками).
*   **Touch Targets:** Минимальная высота интерактивных элементов — `dp(48)` (для удобства нажатия пальцем).

---

## 5. Иконки (Icons)

*   **Формат:** PNG (с прозрачностью).
*   **Стиль:** Line/Outline или Simple Filled.
*   **Расположение:** `assets/icons/`.
*   **Цвет:** Иконки должны быть либо заранее окрашены, либо (предпочтительно) перекрашиваться программно через свойство `color` виджета `Image`.

---

## 6. Пример реализации (Kivy KV)

Пример того, как стили применяются в коде:

```yaml
<RoundedButton>:
    background_normal: ''
    background_color: 0, 0, 0, 0  # Отключаем стандартный фон
    color: 1, 1, 1, 1             # Белый текст
    font_size: '18sp'
    bold: True
    canvas.before:
        Color:
            rgba: 0.15, 0.55, 0.9, 1  # Primary Blue
        RoundedRectangle:
            pos: self.pos
            size: self.size
            radius: [dp(14)]
```

## 7. Анимации и переходы

*   **Screen Transition:** Стандартный слайд (`SlideTransition`) или Fade.
*   **Feedback:** Изменение прозрачности или цвета при нажатии (`state: 'down'`).
