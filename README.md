Kivy Light Quiz App

Этот проект показывает базовую светлую тему и простой тест с результатом.

## Структура
- `main.py` — основной код приложения, включая ScreenManager, тестовую логику и дружелюбную разметку/KV.
- `requirements.txt` — зависимости (`kivy==2.2.1`).
- `buildozer.spec` — заготовка конфигурации для Android.
- `assets/` — иконки, пресплеш и подобное.
- `check_kivy.py` — простой скрипт для проверки, что Kivy установлена.

## Как запустить (Windows)
1. Создайте виртуальное окружение и установите зависимости:

```powershell
cd e:\context_study
python -m venv .venv
. .\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2. Запустите приложение:

```powershell
python main.py
```

## Как пользоваться тестом
1. Нажмите «Начать тест». Откроется экран с вопросом и вариантами ответов (закруглённые карточки).
2. Просто нажмите на любой вариант — ответ оценится **автоматически** и покажет результат.
3. Через 1.5 секунды приложение **автоматически перейдёт** к следующему вопросу.
4. После последнего вопроса откроется экран `Результат` со счётом правильных ответов.
5. На финальном экране можно пройти тест заново или вернуться на главную.
6. На финальном экране можно пройти тест заново или вернуться на главную.

## Дополнительно

	```powershell

	```
